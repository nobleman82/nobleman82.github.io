"Ein hochmodernes, responsives Portfolio-Dashboard, entwickelt mit Blazor WebAssembly und C#. Die Anwendung kombiniert ein minimalistisches 'App-Window'-Design mit dynamischen Layout-Komponenten.

Highlights:

Tech-Stack: ASP.NET Core Blazor WASM, Bootstrap 5 & Custom CSS.

Responsive Design: Optimiert für Ultra-Wide Monitore durch ein flexibles Flexbox-Grid und clamp()-Typografie.

Integration: Dynamische Anbindung von GitHub-Projekten und Highlights aus den Bereichen Softwareentwicklung, Hardware-Prototyping und Musik.

UI/UX: Fokus auf 'Glassmorphism'-Elemente und flüssige Animationen."
